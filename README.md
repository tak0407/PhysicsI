# 물리 놀이터 ⚡

수식으로만 보던 물리 법칙을 직접 만지고, 던지고, 돌려보는 곳.

**👉 https://tak0407.github.io/PhysicsI/**

일반 대중을 위한 한국어 인터랙티브 물리 시뮬레이션 사이트입니다.
모든 실험은 실제 물리 방정식을 실시간으로 적분합니다 — 그럴듯한 애니메이션이 아니라
보존량과 이론값을 화면에서 직접 확인할 수 있습니다.

## 실험 목록 (22개)

| 카테고리 | 실험 |
|---|---|
| 운동과 힘 | 🚀 포물선 운동 · ⛰️ 경사면과 마찰 · 💥 충돌과 운동량 |
| 진동 | 🕰️ 단진자 · 🌀 용수철 · 🌪️ 이중 진자(카오스) · 🎡 공명 |
| 파동과 소리 | 🚑 도플러 효과 (Web Audio 실제 소리) · 🎸 정상파와 배음 |
| 유체와 회전 | 🌊 부력 · ⚖️ 시소와 돌림힘 |
| 전자기 | 🧲 자기장 속 하전입자 · ⚡ 전기장 놀이터 |
| 열역학 | 🔥 기체 분자 운동 (PV=NkT) · ⏳ 엔트로피와 시간의 화살 |
| 우주 | 🪐 행성 궤도 (케플러 법칙) |
| 상대성 | ⏱️ 시간 지연 (빛 시계) |
| 양자 | ⚛️ 이중슬릿 · 📦 상자 속 입자 · 🧱 터널링 (실시간 슈뢰딩거) · 💡 광전 효과 |
| 자유 놀이 | 🧸 물리 놀이터 (Matter.js 샌드박스) |

## 개발

```bash
npm install
npm run dev     # http://localhost:5173
npm run build   # dist/
```

Vite + React 18 + TypeScript. `main` 브랜치에 푸시하면 GitHub Actions가 GitHub Pages로
자동 배포합니다. 프로젝트 구조와 기여 방법은 [AGENTS.md](AGENTS.md), 앞으로의 계획은
[ROADMAP.md](ROADMAP.md)를 보세요.
