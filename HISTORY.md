## 2026-07-04 — 프로젝트 탄생: 빈 폴더에서 18개 실험까지 (첫 세션)

### 시작과 방향 결정

"물리학 법칙을 인터랙션할 수 있는 사이트"라는 아이디어에서 출발.
사용자와 합의한 방향:

- **대상**: 일반 대중 (학생·전공자 아님) → 수식 최소화, 직관적 조작 우선
- **범위**: 역학(Physics I)에서 시작해 점차 확장
- **기술 스택**: 하이브리드 — 법칙 시연은 React + Canvas에 물리 공식을 직접 적분
  (정확한 값, 슬라이더-변수 대응), 자유 놀이만 Matter.js 물리 엔진.
  "엔진은 그럴듯함이 목표라 특정 법칙을 정확히 보여주기엔 오히려 불리하다"는 판단

### 골격 구축 (`6d05cb1`)

- Vite + React 18 + TypeScript(strict) 수동 스캐폴딩 (create-vite가 비대화 환경에서 취소됨)
- 공용 인프라: `useAnimationCanvas` 훅(rAF 루프, dpr 보정, dt 상한 0.05s로 탭 전환 폭주 방지),
  `Slider`/`Readout` 컴포넌트, 다크 테마 CSS 변수
- 첫 실험 5종: 포물선 운동, 단진자(드래그 가능), 1D 충돌(반발 계수),
  행성 궤도(velocity Verlet — 오래 돌아도 에너지 유지), Matter.js 놀이터
- 발견한 문제: 행성 궤도가 한 바퀴에 ~1분으로 너무 느림 → GM 스케일 100배 상향

  핵심 기능은 관측 토글 — 켜면 간섭무늬가 두 줄 분포로 바뀜. 히스토그램+이론 곡선 오버레이

- **터널링**: 슈뢰딩거 방정식을 실시간 수치적분 (560칸 격자, Visscher leapfrog,

  재실행으로 해결. 이후 배포 대기 스크립트에 자동 재시도를 넣음
